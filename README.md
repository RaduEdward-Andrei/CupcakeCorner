# CupcakeCorner

SwiftUI app for creating cupcake orders and submitting them to a backend using async/await networking.

## Overview

CupcakeCorner allows users to configure cupcake orders by selecting type, quantity, and additional options, then submit the order to a remote server. The app focuses on data encoding, networking, and form handling.

## Tech

- Swift
- SwiftUI
- URLSession

## Features

- Create cupcake orders (type, quantity, special requests)
- Encode order data using JSON
- Submit orders to a backend using async/await
- Handle server responses and display feedback to the user
